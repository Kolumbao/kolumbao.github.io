Kolumbao

Kolumbao is a Discord bot that connects guilds with shared channels.

Features

Coming soon

Community search

Using the upcoming kb!search command, you'll be able to search through the possible channels that Kolumbao provides.

Coming soon

Custom channels

Alongside the new kb!search command, you'll be able to create custom channels, either public or private password-protected channels. A way for two servers to connect, no hassle, no fees!

Helpful setup

Never worry about starting using the bot, with our automatic guide and help message, you'll learn how to use the bot and how to set it up. However, if you ever need our help, our support server is available to give you a helping hand

Variety

Kolumbao has a variety of channels, topics and supported languages (you can even translate into your own language). New topics are available by suggestion and a feature to create personalised private groups is coming soon.

Explanation and rules

Using the kb!explanation and kb!rules command, you can see the explanation for Kolumbao and the rules of the channel you're currently in.
